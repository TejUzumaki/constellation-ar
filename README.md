# 🌌 Constellation AR

WebAR application that helps users find and visualize constellations in the night sky.

## Features
- Point your phone at the sky to find constellations
- Cinematic star reveal with connecting lines
- Navigation arrows to guide you to the target constellation
- Real-time astronomical calculations based on your GPS location

## Tech Stack
- Three.js for 3D rendering
- Astronomy Engine for celestial calculations
- WebXR Device API for AR camera access
- Device Orientation API for compass tracking
- Geolocation API for user positioning

## Setup
1. Clone the repository
2. Open `index.html` in a WebXR-compatible browser
3. Allow camera, location, and motion permissions
4. Look up at the stars!
